# 📝 To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 https://sundramd7.github.io/todo-list-app/

## 📌 Features
- Add new tasks
- Mark tasks as completed with strike-through
- Delete tasks
- Data stored using Local Storage

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Learning Outcomes
- DOM Manipulation
- Event Handling
- Local Storage
- Basic Frontend Development
